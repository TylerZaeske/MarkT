# Available Tags

```css
text-shadow: 0 0 5px currentColor;
```

## unglow

```css
text-shadow: none;
```

```css
font-weight: bold;
```

```css
font-style: italic;
```

```css
text-decoration: underline;
```

```css

```

```css
font-size: smaller;
```

```css
vertical-align: sub;
font-size: smaller;
```

## small

```css
font-size: smaller;
```

## large

```css
font-size: larger;
```
